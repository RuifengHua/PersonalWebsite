Sorting Algo Visualizer

An online sorting algorithm visualizer using React and JavaScript where users are able to generation small/median/large sized array and visualize it by the most common used sorting algorithms including merge sort, quick sort and etc…

Users are also able to see the detailed information such as time complexity, pseudo-code and high level idea of each sorting algorithm for educational purpose.
